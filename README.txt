# IRF_Project
Indításkor egy gól látható, így magunkhoz vonzva a szerencsét!
A program az Arsenal focicsapat góllövőit követi nyomon. Az adatbázisban táruljuk a góllövőket. A program segít változtatni a játékos állományon.
Található egy diagram, aktuális góllövőkkel és gólpassz adókkal, ugyanis a gólt nem csak a góllövő szerzi, hanem az is aki neki adja a passzt.
